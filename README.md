# Aeris

Preventing mold. Protecting health.

Aeris is a student-led public health initiative distributing low-cost mold-prevention kits to renters in Philadelphia, targeting the 24 to 48 hour window before mold takes hold after a water leak.

**Live site:** https://theaerisinitiative.org (custom domain in progress) / https://yourusername.github.io/your-repo-name

## What's in this repo

This is a single-page static website, no build step, no dependencies, no backend server.

- `index.html`, the entire site (HTML, CSS, and JS in one file)

## Tech

- Plain HTML/CSS/JS
- Fonts: Archivo Black, Work Sans, IBM Plex Mono (loaded from Google Fonts)
- Forms: [FormSubmit](https://formsubmit.co) handles the "Request a kit," "Become a partner," and "Contact us" forms with no backend or API key required. Submissions email to aerisinitiative@gmail.com.
- Hosting: GitHub Pages

## Making changes

1. Edit `index.html` directly on GitHub (pencil icon on the file), or edit locally and re-upload it
2. Commit to the `main` branch
3. GitHub Pages rebuilds automatically, changes usually go live within a minute or two

## Notes

- The kit request form requires checking three eligibility boxes: renter status with a minimum 30-day residency, a recent leak or moisture issue, and a household income at or below $100,000/year.
- FormSubmit requires a one-time confirmation the first time a form is submitted to a new email address, check the inbox and click the confirmation link if submissions aren't arriving.
